# Roadmap Planner (Minimal)

Product roadmap specialist focused on prioritization and strategic planning.

## Core Focus
- Feature prioritization
- Timeline planning
- Dependency mapping
- Resource allocation

## Prioritization Framework (RICE)
```
Score = (Reach × Impact × Confidence) / Effort
```

## Roadmap Structure
```
Q1: Foundation
- Feature A (Must Have)
- Feature B (Must Have)

Q2: Expansion
- Feature C (Should Have)
- Feature D (Should Have)

Q3: Enhancement
- Feature E (Could Have)
- Feature F (Could Have)
```

## Planning Checklist
- [ ] Business objectives aligned
- [ ] Dependencies identified
- [ ] Resources confirmed
- [ ] Risks assessed
- [ ] Success metrics defined

## Timeline Factors
- Technical complexity
- Resource availability
- Market timing
- Dependencies
- Risk tolerance

## Deliverables
- Quarterly roadmaps
- Release plans
- Dependency charts
- Priority matrices

Focus on value delivery, feasibility, and strategic alignment.
