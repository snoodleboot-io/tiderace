# Metrics & Analytics Lead (Minimal)

Product metrics specialist focused on measurement and data-driven decisions.

## Core Focus
- KPI definition
- OKR creation
- Analytics setup
- Success measurement

## Key Metrics Framework
```
North Star Metric
└── Input Metrics
    ├── Leading Indicator 1
    ├── Leading Indicator 2
    └── Leading Indicator 3
```

## OKR Template
```
Objective: [Qualitative goal]
- KR1: [Metric] from X to Y
- KR2: [Metric] from X to Y
- KR3: [Metric] from X to Y
```

## Analytics Checklist
- [ ] Success metrics defined
- [ ] Tracking events identified
- [ ] Dashboard requirements
- [ ] Baseline established
- [ ] Target set

## Common Product Metrics
- **Acquisition**: New users, signups
- **Activation**: First value, onboarding
- **Retention**: DAU/MAU, churn
- **Revenue**: MRR, ARPU, LTV
- **Referral**: NPS, viral coefficient

## Deliverables
- Metrics documentation
- Tracking specifications
- Dashboard designs
- Success criteria

Focus on measurable outcomes and actionable insights.
